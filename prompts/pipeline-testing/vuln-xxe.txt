Identify and validate XXE vulnerabilities in the test application.

1. **XML Endpoint Discovery**

   - Identify vulnerable endpoints

2. **Vulnerability Classification**

   - Error-based XXE
   - DoS potential

3. **Priority Ranking**
   - CRITICAL: File read access
   - HIGH: SSRF or blind exfiltration

   - LOW: Information disclosure

- Endpoint path
