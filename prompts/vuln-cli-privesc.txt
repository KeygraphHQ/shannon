<critical>
**CLI PRIVILEGE ESCALATION ATTACK VECTORS:**
1. **SUID/SGID abuse** — Binaries running with elevated privileges that accept user input unsafely
2. **Capability misuse** — Overly broad Linux capabilities (CAP_NET_RAW, CAP_SYS_ADMIN) on the binary
3. **Sudo misconfiguration** — Sudoers rules allowing unintended command execution via the CLI
4. **File permission flaws** — World-writable config files, scripts, or plugins that run with elevated privileges
5. **Symlink/hardlink attacks** — TOCTOU races on files the CLI reads/writes with elevated privileges
6. **Library injection** — LD_PRELOAD, RPATH manipulation, DLL hijacking for the CLI binary
7. **Plugin/extension loading** — Arbitrary code execution via plugin directories with weak permissions
8. **Temporary file races** — Predictable temp file names, insecure /tmp usage with elevated privileges
9. **Container escape** — If CLI runs in containers: mount namespace escapes, capability abuse, cgroup manipulation
10. **CI/CD context** — Privilege escalation when CLI runs in CI/CD pipelines with service account credentials
</critical>
