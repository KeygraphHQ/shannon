<critical>
**BOLA/IDOR ATTACK VECTORS:**
1. **Direct object reference** — Incrementing/decrementing numeric IDs in URLs or request bodies
2. **UUID enumeration** — Predictable UUIDs (v1 time-based), leaked UUIDs in other responses
3. **Nested resource access** — `/users/{id}/orders/{order_id}` — checking user ID but not order ownership
4. **Batch/bulk operations** — Mass data access via list endpoints without proper filtering
5. **GraphQL traversal** — Following relationships to access unauthorized objects
6. **File/media access** — Direct URL access to files belonging to other users
7. **Webhook/callback manipulation** — Changing notification targets to attacker-controlled endpoints
8. **Pagination abuse** — Accessing out-of-scope data via crafted pagination parameters
</critical>
